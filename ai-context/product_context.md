## Vision

Build an AI-powered resume optimization platform that helps job seekers create ATS-compatible resumes that maximize their chances of passing automated screening systems while maintaining authenticity and professionalism.

## Target Users

### Primary: Job Seekers
- Career changers needing resume reformatting
- Recent graduates with limited resume experience
- Professionals applying to multiple positions
- International candidates unfamiliar with ATS systems

### Secondary: Career Professionals
- Resume writers wanting AI assistance
- HR consultants helping clients
- Career coaches

## Core Features (MVP)

1. **Resume Upload & Parsing**
   - PDF/DOCX upload
   - Structured data extraction
   - Section identification

2. **Job Description Analysis**
   - Keyword extraction
   - Requirements parsing
   - Skill categorization

3. **ATS Score Calculation**
   - Resume-to-job matching
   - Keyword coverage score
   - Format compatibility check

4. **AI-Powered Suggestions**
   - Missing keyword recommendations
   - Phrasing improvements
   - Section optimization

5. **Resume Builder**
   - ATS-friendly templates
   - Real-time editing
   - Export to PDF/DOCX

## Success Metrics

| Metric | Target |
|--------|--------|
| ATS score accuracy | >85% correlation with real ATS |
| User satisfaction | >4.5/5 stars |
| Resume improvement | >20 point score increase |
| Parse accuracy | >95% for standard resumes |
| Response time | <3 seconds for scoring |

## Constraints

- Never fabricate experience
- Never claim skills user doesn't have
- Preserve user's authentic voice
- Support multiple languages (future)
- GDPR/privacy compliant
